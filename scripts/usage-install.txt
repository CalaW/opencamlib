Usage: ./scripts/install-PLATFORM.sh [OPTIONS] LIB BUILD_TYPE

Options:
  --boost-prefix          set a custom path where to look for Boost
  --boost-from-source     compile Boost from source
    --boost-address-model   set the address model for Boost (only for compile)
    --boost-architecture    set the architecture for Boost (only for compile)
    --python-executable     set a custom path to the Python executable (only for compile)
  --no-deps               skip installing dependencies
  --help                  shows this page
Libraries:
  cxxlib | nodejslib | python3lib | emscriptenlib
Build types:
  debug | release
